# mooda

Current version: 0.0.1-dev

Multi-Objective Optimisation algorithm for sequence Design and Assembly

## Installation

    pip install mooda

## Documentation

The official documentation for mooda can be found on [readthedocs](https://mooda.readthedocs.io/).
